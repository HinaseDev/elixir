class WSGI(builtins.object)
    WSGI(handler) -&gt; None
 
Most basic Handler for WSGI Servers.
 
Sample Usage:
 
```py
from elixir.application import WSGI, wrap_response
from elixir.serving import run_server
 
def recv(request, response):
    return wrap_response("Hello World", response)
 
run_server(WSGI(recv))
```

wrap_response(text, start_response, *, headers: elixir.middleware.Headers = <elixir.middleware.Headers object at 0x7fa49f1b6c20>, status_code='200 Ok', mimetype='text/html')
The wrap_response function takes a string and returns an iterable that can be passed to the WSGI server.
The function also accepts keyword arguments, which are used to set up the response headers.
 
 
:param text: Pass the body of the response
:param start_response: Set the status code and headers for the response
:param *: Pass a variable number of arguments to a function
:param headers:Headers=InitialHeaders: Set the default headers for all responses
:param status_code=&quot;200Ok&quot;: Specify the status code of the response
:param mimetype=&quot;text/html&quot;: Specify the type of content that is being returned
:return: The response text, the status code and the headers
